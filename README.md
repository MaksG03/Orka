## Про проект

Orka — це інтелектуальний конвертер розкладки клавіатури, що працює локально та підтримує наднизьку затримку (<100мс). Додаток збудований на **Rust** (повністю переписаний з C++) і забезпечує безшовну інтеграцію як на Windows, так і на Linux через системний трей та нативні APIs.

Orka підтримує мовні пари:
* **EN ↔ UK** (Англійська ↔ Українська)
* **EN ↔ KO** (Корейська — Dubeolsik)
* **EN ↔ HE** (Іврит — з обробкою Nikud)

## Використання

1. Запустіть додаток (з'явиться іконка в системному треї).
2. Виділіть невірно набраний текст у будь-якому додатку (наприклад, `ghbdim`).
3. Натисніть **Alt+Z**.
4. Orka автоматично визначить напрямок (латиниця чи кирилиця) та замінить текст (наприклад, на `привіт`).

## Карта символів (Українська)

Конвертер використовує O(1) мапу позицій клавіатури. Враховуються:
- Всі літери верхнього/нижнього регістру
- Спеціальні українські літери: `Ґ` (`~`), `Х` (`{`), `Ї` (`}`), `Ж` (`:`), `Є` (`"`), `Б` (`<`), `Ю` (`>`)
- Автоматична заміна друкарських лапок (`" " ↔ « »`)

## Особливості
* **Кросплатформеність:** Нативна підтримка Windows (Microsoft Store MSIX) та Linux (.deb пакети).
* **Rust Backend & React Frontend:** Надвисока швидкодія за рахунок Rust; гарний UI налаштувань на базі React через Webview.
* **Швидкість:** Конвертація відбувається локально з використанням `HashMap` з доступом за O(1).
* **Автовизначення:** Конвертер розуміє, який алфавіт переважає у виділеному тексті, і перетворює його в протилежний.

## Build

Для самостійної збірки проекту встановіть [Rust/Cargo](https://rustup.rs/) та [Node.js](https://nodejs.org/).

```bash
# 1. Зібрати UI
cd src/ui/webapp
npm install
npm run build
cd ../../..

# 2. Зібрати Orka
cargo build --release
```

Двійковий файл буде знаходитись у `target/release/orka`.
